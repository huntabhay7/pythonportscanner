# python portscanner

portscanner is LAN scanner in python 3

It can be used to scan a target or a network range

it works on ARP to scan network

Installation
git clone 

cd portscanner

sudo chmod +x setup.sh

sudo bash setup.sh

portscanner -t

Example
portscanner -t 192.168.1.1

portscanner -t 192.168.1.0/24
